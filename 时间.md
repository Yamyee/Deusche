|--------|----------|
| die Zeit | time     |
| die Stunde | hour     |
| die Minute | minute   |
| die Sekunde | second   |
| die Uhr | clock/clock time   |
| die Mitternacht | midnight |
| der Morgen | morning  |
| der Vormittag | forenoon  |
| der Nachmittag | afternoon |
| der Abend | evening   |
| die Nacht | night     |

| 德语    | 英语     |
|---------|----------|

| der Sommer | summer   |
| der Herbst | autumn/fall |
| der Winter | winter   |
